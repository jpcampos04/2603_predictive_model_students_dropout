# 🎓 Student Dropout Prediction (2024)

## 📌 Project Overview
This project analyzes a dataset provided by an online learning platform containing information about students enrolled in courses during 2024. The dataset includes geographic, demographic, and behavioral variables, with the primary goal of building a machine learning model capable of predicting student dropout in online courses.

The project combines exploratory data analysis (EDA) and predictive modeling to generate actionable insights and improve student retention strategies.
